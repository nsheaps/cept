# Cept vs Obsidian

Cept and Obsidian share a philosophy of local-first, Markdown-based note-taking. Here's how they compare.

## Quick Comparison

| Feature | Cept | Obsidian |
|---------|------|----------|
| **Pricing** | Free, open source (MIT) | Free for personal use, paid for sync/publish |
| **Open source** | Yes (MIT) | No (proprietary, source-available) |
| **Data format** | Markdown + YAML | Markdown |
| **Storage** | Browser, Local, or Git | Local folder |
| **Web app** | Yes (PWA) | No native web app |
| **Block editor** | WYSIWYG blocks (Notion-style) | Live preview Markdown |
| **Databases** | Built-in (6 views, 18 types) | Via Dataview plugin |
| **Knowledge graph** | Built-in | Built-in |
| **Templates** | Built-in | Built-in + Templater plugin |
| **Collaboration** | Git-based | Obsidian Sync (paid) |
| **Plugins** | Not yet | 1000+ community plugins |
| **Version history** | Git log (unlimited) | Obsidian Sync (paid) |
| **Search** | Built-in full-text | Built-in full-text |
| **Mobile** | Coming soon | Available |
| **Self-hosting sync** | Yes (any Git remote) | No (Obsidian Sync is proprietary) |

## Where Cept Wins

### Web Access
Cept runs in any browser as a PWA. No installation required. Obsidian requires downloading a desktop app and doesn't have a native web version.

### Databases
Cept has built-in structured databases with table, board, calendar, gallery, list, and map views — similar to Notion. Obsidian relies on the Dataview plugin for queries, which is powerful but requires learning a query language.

### Block Editor
Cept's WYSIWYG block editor works like Notion — click, type, drag blocks. Obsidian uses a Markdown-focused live preview that's powerful but has a steeper learning curve for non-technical users.

### Zero Setup
Cept can run entirely in the browser with no installation. Just open the URL and start writing. Obsidian requires downloading and installing the app, then choosing a vault location.

### Git-Native Sync
Cept uses standard Git for sync and collaboration. Any Git host works (GitHub, GitLab, Gitea, self-hosted). Obsidian's sync is a proprietary paid service.

### Open Source
Cept is fully MIT-licensed open source. Obsidian is proprietary (though the source is available for review).

## Where Obsidian Wins

### Plugin Ecosystem
Obsidian has 1000+ community plugins covering everything from Kanban boards to spaced repetition. Cept doesn't have a plugin system yet.

### Maturity
Obsidian is battle-tested with a large, active community. Cept is new.

### Desktop Performance
Obsidian is optimized for desktop with native-feeling performance. Cept's desktop apps are still in development.

### Mobile Apps
Obsidian has mature mobile apps. Cept's mobile support is coming soon.

### Canvas
Obsidian Canvas provides an infinite whiteboard for spatial thinking. Cept doesn't have this yet.

### Community
Obsidian has a large community with forums, Discord, and extensive documentation. Cept is just starting to build its community.
