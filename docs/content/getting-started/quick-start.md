# Quick Start

Get up and running with Cept in under a minute.

## Web (Fastest)

1. Visit the Cept web app at your hosted URL (or the GitHub Pages demo)
2. Click **Start writing** — you'll be dropped into a new space using browser storage
3. Start creating pages with the block editor

No account required. Everything is stored locally in your browser using localStorage.

To try the demo without affecting your data, add `?demo` to the URL.

## Desktop

1. Download Cept for your platform (macOS, Windows, or Linux)
2. Open the app
3. Choose a storage mode:
   - **Browser storage** for quick start
   - **Local folder** to store files on your computer (coming soon)
   - **Git repository** for version control and sync (coming soon)

## Mobile

1. Install Cept from the App Store or Google Play (coming soon)
2. Open the app and start writing
3. Data persists locally on your device

## Creating Your First Page

1. Click the **+** button in the sidebar to create a new page
2. Click the page title to rename it
3. Type `/` in the editor to open the slash command menu
4. Choose a block type (heading, list, code, callout, etc.)
5. Keep typing — content auto-saves as you edit

## Organizing Pages

- **Nested pages** — Click the `+` on any page in the sidebar to create a sub-page
- **Favorites** — Right-click (or tap the `...` menu) on a page and choose "Add to favorites"
- **Trash** — Delete a page to move it to trash. Restore it anytime from the trash section in the sidebar.
- **Drag and drop** — Reorder pages by dragging them in the sidebar (coming soon)

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `/` | Slash command menu |
| `Cmd/Ctrl + B` | Bold |
| `Cmd/Ctrl + I` | Italic |

See the full [keyboard shortcuts reference](../reference/keyboard-shortcuts.md).

## Connect to Git (Optional)

To enable version history and sync:

1. Open **Settings** from the app menu (top-right `...` button)
2. Go to the **Spaces** tab
3. Create or connect a Git-backed space
4. Authenticate with GitHub (or enter any Git remote URL)
5. Your space will sync automatically
