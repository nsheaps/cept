# Quick Start

Get up and running with Cept in under 5 minutes.

## Web (Fastest)

1. Visit the Cept web app
2. Click "Get Started" — you'll be dropped into a new workspace using browser storage
3. Start creating pages and databases

No account required. Everything is stored locally in your browser.

## Desktop

1. Download Cept for your platform (macOS, Windows, or Linux)
2. Open the app
3. Choose a storage mode:
   - **Browser storage** for quick start
   - **Local folder** to store files on your computer
   - **Git repository** for version control and sync

## Mobile

1. Install Cept from the App Store or Google Play
2. Open the app and start writing

## Connect to Git (Optional)

To enable version history and sync:

1. Open Settings > Storage
2. Click "Connect Git Repository"
3. Authenticate with GitHub (or enter any Git remote URL)
4. Select or create a repository
5. Your workspace will sync automatically
