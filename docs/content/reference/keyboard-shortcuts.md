# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + N` | New page |
| `Cmd/Ctrl + S` | Save (manual sync) |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + F` | Search in page |
| `Cmd/Ctrl + Shift + F` | Global search |

## Editor

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold |
| `Cmd/Ctrl + I` | Italic |
| `Cmd/Ctrl + U` | Underline |
| `Cmd/Ctrl + Shift + S` | Strikethrough |
| `Cmd/Ctrl + E` | Inline code |
| `Cmd/Ctrl + Shift + H` | Highlight |
| `/` | Slash command menu |
| `---` | Horizontal divider |
| `>` + Space | Blockquote |
| `#` + Space | Heading 1 |
| `##` + Space | Heading 2 |
| `###` + Space | Heading 3 |
| `-` or `*` + Space | Bullet list |
| `1.` + Space | Numbered list |
| `[]` + Space | To-do item |
| `` ``` `` + Enter | Code block |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + P` | Quick page switch |
| `Cmd/Ctrl + [` | Go back |
| `Cmd/Ctrl + ]` | Go forward |
| `Cmd/Ctrl + \` | Toggle sidebar |

## Database

| Shortcut | Action |
|----------|--------|
| `Tab` | Next cell |
| `Shift + Tab` | Previous cell |
| `Enter` | Edit cell |
| `Escape` | Exit cell edit |
